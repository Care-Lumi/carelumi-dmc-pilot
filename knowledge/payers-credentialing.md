# Payers & Credentialing Page

## Overview
The Payers page manages credentialing requirements, applications, and enrollment status with insurance companies and government payers.

## Payer List View

### Displayed Information
- **Payer Name**: Insurance company name
- **Payer Type**: Commercial, Medicare, Medicaid, Other
- **Enrolled Providers**: Count of credentialed providers
- **Pending Applications**: In-progress applications
- **Requirements**: Number of required documents
- **Average Processing Time**: Historical data
- **Status**: Active, Pending Setup, Inactive

### Filters
- Payer Type
- Status
- Processing Time
- Number of Enrolled Providers
- Facility

## Payer Detail Page

### General Information
- Payer Name
- Payer ID
- Type (Commercial, Medicare, Medicaid)
- Contact Information
- Provider Services Phone
- Website
- Portal URL
- Application Portal Login

### Credentialing Requirements
- Required Documents Checklist:
  - Application Form
  - CV/Resume
  - Medical License (with specific requirements)
  - DEA Certificate
  - Board Certification
  - Malpractice Insurance (coverage amounts)
  - Hospital Privileges
  - Education Verification
  - Work History
  - References
  - Immunization Records
  - Health Screening
  - Background Check
  - CAQH Profile

### Processing Timeline
- Application Submission
- Initial Review (typical: 7-14 days)
- Request for Additional Information
- Committee Review (typical: 30-45 days)
- Final Approval
- Effective Date
- Re-credentialing Cycle (typically 3 years)

### Provider Enrollment Status
Table showing all providers with this payer:
- Provider Name
- Application Status
- Submission Date
- Expected Completion Date
- CAQH Status
- Missing Items
- Last Updated
- Actions (View, Update, Contact Payer)

### Application Status Tracking
- **Not Started**: No application submitted
- **In Progress**: Application being prepared
- **Submitted**: Application sent to payer
- **Under Review**: Payer reviewing application
- **Additional Info Requested**: Need more documents
- **Committee Review**: Awaiting committee decision
- **Approved**: Provider credentialed
- **Denied**: Application rejected
- **Re-credentialing Due**: Time for renewal

## Adding Provider to Payer

### Step 1: Select Provider
- Choose from existing providers
- View provider's complete credential package
- Check readiness score

### Step 2: Application Type
- Initial Application
- Re-credentialing
- Address Change
- Practice Location Addition
- Specialty Addition

### Step 3: Required Documents
- Checklist of payer requirements
- Upload or select existing documents
- Verify document currency
- Flag missing items

### Step 4: CAQH Integration
- Link to provider's CAQH profile
- Authorize payer access
- Verify CAQH status (Attestation Current)
- Set re-attestation reminders

### Step 5: Application Review
- Review completeness
- Check for errors
- Validate all dates
- Confirm contact information

### Step 6: Submit
- Electronic submission (if available)
- Generate submission packet
- Track confirmation
- Set follow-up reminders
