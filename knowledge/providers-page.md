# Providers Page Walkthrough

## Overview
The Providers page is your central hub for managing provider credentialing, licenses, certifications, and payer enrollment status.

## Provider List View

### List Columns
- **Provider Name**: Full name with photo
- **Specialty**: Primary specialty
- **Status**: Active, Inactive, Pending
- **Licenses**: Count and status indicator
- **Certifications**: Board certifications status
- **Payer Enrollment**: Number of active payers
- **Next Expiration**: Soonest expiring credential
- **Actions**: Quick action menu

### Filters
- Status (Active/Inactive/Pending)
- Specialty
- Facility
- Credentialing Status
- Expiration Date Range
- Payer Enrollment Status

### Bulk Actions
- Export Selected
- Send Reminder Emails
- Generate Reports
- Assign to Payer
- Update Status

## Adding a New Provider

### Step 1: Basic Information
- First Name, Last Name
- NPI Number (validates in real-time)
- Date of Birth
- Email Address
- Phone Number
- Specialties (primary and secondary)

### Step 2: Employment Details
- Facility Assignment
- Employment Start Date
- Employment Type (Full-time, Part-time, PRN)
- Department
- Supervisor

### Step 3: License Information
- State License Number
- Issue Date
- Expiration Date
- License Type
- Upload License Document
- Set expiration reminders (30, 60, 90 days)

### Step 4: Certifications
- Board Certification Type
- Certifying Body
- Issue Date
- Expiration Date (if applicable)
- Certificate Number
- Upload Certificate

### Step 5: Insurance
- DEA Registration
- Malpractice Insurance
- Policy Number
- Coverage Amounts
- Expiration Date
- Upload Proof of Insurance

## Provider Detail Page

### Tabs
1. **Overview**: Summary of all credentials and status
2. **Documents**: All uploaded documents organized by type
3. **Payer Enrollment**: Status with each payer
4. **Facilities**: Assigned facilities and privileges
5. **History**: Audit trail of all changes
6. **Attestations**: Required attestations and signatures

### Document Management
- Drag and drop upload
- Automatic OCR to extract dates
- Document categorization
- Version history
- Expiration tracking
- Reminder settings

### Payer Enrollment Tracking
- Application status for each payer
- CAQH status integration
- Application dates
- Expected completion dates
- Required documents checklist
- Communication log

## Expiration Management
- Automatic reminders at 90, 60, 30 days
- Email notifications to provider and admin
- Dashboard alerts
- Color-coded status indicators
- Bulk renewal tools

## Compliance Alerts
- Missing Required Documents
- Expired Credentials
- Expiring Soon (within 90 days)
- Incomplete Applications
- Attestation Needed
