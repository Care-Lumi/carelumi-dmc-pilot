# CareLumi Dashboard Overview & Navigation

## Main Dashboard Components

### Header Navigation
- **Organization Selector**: Switch between facilities/organizations
- **Search Bar**: Global search across providers, payers, documents
- **Notifications Bell**: Real-time alerts for expirations, compliance items
- **User Profile**: Account settings, logout

### Left Sidebar Menu
- Dashboard (Home)
- Providers
- Payers
- Facilities  
- Staff
- Documents
- Audit Readiness
- Revenue Risk
- Regulatory Updates
- Reports
- Settings

### Dashboard Cards

**Credentialing Pipeline**
- Shows providers in different stages: Initial, In Progress, Expiring Soon, Complete
- Click any stage to see provider list
- Visual progress indicators
- Quick action buttons

**Critical Alerts**
- Urgent items requiring immediate attention
- Color-coded by severity (red=critical, yellow=warning)
- Direct links to resolve items
- Real-time updates

**Expiring Soon**
- Documents/credentials expiring within 90 days
- Sortable by date, provider, type
- Bulk actions available
- Export capabilities

**Audit Readiness Score**
- Overall compliance percentage
- Breakdown by category
- Trend over time
- Action items to improve

**Revenue at Risk**
- Financial impact of compliance gaps
- Broken down by payer
- Projection of potential losses
- Priority recommendations
