# Documents & Compliance

## Overview
The Documents page is your centralized repository for all compliance documents, policies, procedures, and regulatory materials.

## Document Library

### Document Organization
Documents are organized by:
- **Category**: Licenses, Certifications, Insurance, Policies, Training, etc.
- **Type**: License, Certificate, Policy, Procedure, Form, Report, etc.
- **Owner**: Provider, Staff, Facility, Organization
- **Status**: Current, Expiring Soon, Expired, Archived

### Document List View
- Document Name
- Category
- Owner
- Upload Date
- Expiration Date (if applicable)
- Status
- Tags
- Actions (View, Download, Edit, Delete)

## Uploading Documents

### Manual Upload
- Click "Upload Document" button
- Select file (PDF, Word, Excel, Images)
- Choose document category
- Assign to owner (provider, staff, facility)
- Set expiration date (if applicable)
- Add tags for easy searching
- Set reminder notifications
- Add notes/description

### Drag and Drop
- Drag files directly into the upload area
- Bulk upload multiple files
- Automatic categorization suggestions
- Quick tag assignment

### Automatic OCR
- Automatically extract key information:
  - Document type
  - Issue date
  - Expiration date
  - License/Certificate numbers
  - Names
- Review and confirm extracted data
- Saves time on data entry

### Email Import
- Forward documents to unique email address
- Automatically added to document library
- Use subject line for categorization
- Attach to correct owner via email tags
