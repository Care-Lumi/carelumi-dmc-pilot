# CareLumi Platform Guide

## Overview
CareLumi is an AI-powered compliance platform designed specifically for Ambulatory Surgery Centers (ASCs) in Texas. The platform helps ASC administrators manage credentialing, licensing, regulatory compliance, and operational workflows.

## Dashboard Overview
The CareLumi dashboard provides a comprehensive view of your facility's compliance status with real-time alerts and actionable insights.

### Key Dashboard Components:
- **Critical Alerts** - Time-sensitive compliance issues requiring immediate attention
- **Credentialing Pipeline** - Visual tracking of provider credentialing status
- **Expiring Soon** - Upcoming license and certification renewals
- **Audit Readiness Score** - Overall compliance health metric
- **Quick Actions** - Common tasks like uploading documents, running audits, and managing staff

## Payers Page
The Payers page is your central hub for managing payer relationships and provider credentialing across different insurance networks.

### Payer Credentialing Overview:
- Track credentialing status for each provider across multiple payers
- Monitor application progress and identify bottlenecks
- Receive alerts for missing documents or pending actions
- View historical credentialing data and renewal timelines

### Key Features:
1. **Payer List View** - See all payer relationships with status indicators
2. **Provider Assignment** - Link providers to specific payer networks
3. **Document Management** - Upload and organize credentialing documents per payer
4. **Status Tracking** - Real-time updates on application progress
5. **Renewal Reminders** - Automated alerts before credential expiration

### Common Payer Actions:
- Add new payer relationships
- Assign providers to payers
- Upload credentialing documents
- Check application status
- Generate credentialing reports
- Set renewal reminders
